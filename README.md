# Tuckshop-Management-System

## Domain
Business and Informal Trade

## Description
This project presents a java based console application that digitises the operations of a small tuckshop by monitoring current stock levels, processing sales transacto-ions and generating a sales summary report as well as a menu that will allow the user to add and categorise products
It is built using OOP principles like inheritance,composition and exception handling

## Features
- Record sales
- Manage stock
- Track products
