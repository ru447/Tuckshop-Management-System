# Tuckshop-Management-System

## Domain
Business and Informal Trade

## Description
This project presents a java based console application that digitises the operations of a small tuckshop by monitoring current stock levels, processing sales transactions and generating a sales summary report as well as a menu that will allow the user to add and categorise products.This project presents a Java-based application that digitises the core operations of a small tuckshop. The system is built around two main entity types — Product and Sale — where Product is an abstract class extended by FoodItem and DrinkItem, demonstrating inheritance and polymorphism. A MainApp class acts as the central domain object, owning collections of products and sales through composition. All business logic is handled by TuckshopService. As well as payment methods for types of payment using inheritance and exceptions to handle errors.

## Features
- Record sales and track products
- Manage stock
- Displays a menu
